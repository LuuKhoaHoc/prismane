## Key Features

- Built on top of Tailwind CSS design system
- Dynamic theming out of the box
- Custom animation components
- Custom hooks

## How To Use

[Read Documentation](https://prismane.io)

```bash
# With yarn
yarn add @prismane/core

# With npm
npm install @prismane/core
```
